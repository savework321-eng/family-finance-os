# Folder Structure

## Overview

PFOS follows a modular folder structure to improve maintainability and scalability.

```
family-finance-os/

docs/

src/
    backend/
    frontend/
    shared/
    tests/

spreadsheet/

assets/

release/
```

---

## Backend

Contains all Google Apps Script source code.

Modules:

- Controllers
- Services
- Repositories
- Utilities
- Config
- Middleware

---

## Frontend

Contains HTML, CSS and JavaScript.

Modules:

- Pages
- Components
- Layouts
- Assets

---

## Shared

Contains reusable code.

Examples:

- Constants
- Helpers
- Validators
- Enums

---

## Tests

Contains unit tests and integration tests.

---

## Assets

Contains static assets.

Examples:

- Images
- Icons
- Logos
